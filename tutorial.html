Tutorial #1

Making a Cydia repo using GitHub Pages

Step 1: GitHub Account

Go to GitHub and Login or Sign-Up



Step 2: New GitHub Repo

Create a new repo with the same name as your account name, Account(antiquedev), Repo(antiquedev)



Step 3: Cloning the Repo

Open up MTerminal or your favourite terminal prompt and login as root(su followed by your password)

Type: git clone https://www.github.com/(your username)/(your username).github.io.git



Step 4: Installing The Files

Go to Dropbox and download the files, after they have downloaded, copy them into /var/mobile/(your username).github.io



Step 5: Editing The Files

Open up "Packages", "Release" and "index.html" and edit to your preference. Inside of index.html you should see a part with ((CHANGE TO YOUR REPO)), just use the same repo as you used prior, ie; username.github.io(remove brackets)



Step 6: Add Tweak Debian

Adding the debian is simply placing your tweak debian(.deb) into the debs folder,

WARNING: Make sure you edit the .deb name to be the same as "Filename" inside of Packages



Step 7: Compress and Compile

Open or go back to your terminal prompt(root permission) and cd back into your /var/mobile/(username.github.io) folder.

ALL ARE LOWER CASE
Type: bzip2 -z Packages(spelt with capital "P" and lowercase "ackages")

Type: git add --all

Type: git commit -m "Initial"

Type: git push

Enter the correct credentials to complete the process and wait for the upload to finish.

Step 8: Connect GitHub Pages

Now we have everything sorted for our repo, we will want to make our website and links go online. to do this we will want to go to GitHub again and go into our created repo, you will see an option called "Settings", click on that and then scroll down until you see "GitHub Pages", this will activate our website. Click the "None" tab and select "master branch" and then press save. That's it!

We have now created a fully working repo using GitHub Pages. Click your site link and you will see the website load(the link should be http://github.com/username.github.io). Click the + icon and you will be directed to Cydia, press Install Source and watch as your repo is added to Cydia.

ENJOY!
